Lesson 3: Introduction to Python

Welcome to Python, one of the most popular and versatile programming languages on the planet! Python is famous for its simple, readable syntax, which makes it an excellent choice for beginners. But don't let its simplicity fool you; it's also incredibly powerful and is used by major companies like Instagram, Netflix, and Spotify for everything from building websites to powering complex data analysis and artificial intelligence.

What Can You Do with Python?

Python is a general-purpose language, meaning it isn't specialized for just one task. You can use it for:

  • Data analysis and machine learning
  • Web development (the "back-end" logic of websites)
  • Automation or scripting to automate repetitive tasks
  • Software testing and prototyping
  • Solving everyday tasks and problems

Getting Started: Your Programming Environment

To start coding in Python, you need two things:

  1. An Interpreter: This is the core program that understands your Python code. It reads your .py files line by line and translates them into instructions the computer can execute. Download and install the official Python interpreter from www.python.org/downloads.
  2. An IDE (Integrated Development Environment): This is the software where you will write your code. It is like a specialized text editor with helpful features such as a debugger and build automation tools. Popular choices include:
    • PyCharm (Community Edition is free)
    • Visual Studio Code (VS Code) (A very popular, free option)
    • IDLE (Comes bundled with the standard Python installation)

Python Fundamentals: The Building Blocks

1. Comments

Comments are notes in your code that are ignored by the Python interpreter. They are used to explain your code and make it more readable for yourself and others. In Python, any text following a hash symbol (#) is a comment.

# This is a comment. It won't be executed.
print("Hello, World!")  # This part of the line is also a comment.

2. Variables and Data Types

Variables are containers for storing data values. You create one simply by assigning a value to a name.

name = "Abebe"  # A variable 'name' of type string
age = 16         # A variable 'age' of type integer

Python has several built-in data types:

  • Text Type (str): A string of text, enclosed in single or double quotes. y = "John" or z = 'Python'
  • Numeric Types:
    • int: An integer (a whole number). x = 5
    • float: A floating-point number (a number with a decimal). price = 99.5
  • Boolean Type (bool): Represents one of two values: True or False. Used for conditional logic. is_active = True
  • Sequence Types: list for ordered collections, tuple, range
  • Mapping Type: dict (dictionary) for key-value pairs

You can check the data type of any variable using the type() function:

x = 5
print(type(x))  # This will display <class 'int'>

3. User Input

Python allows you to get input from the user during program execution using the input() function.

username = input("Enter your username: ")
print("Hello, " + username)

Control Flow: Making Decisions and Repeating Actions

1. Conditional Statements (if, elif, else)

These statements allow your program to execute different blocks of code based on whether a condition is true or false.

a = 33
b = 200
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")

2. Loops (while and for)

Loops are used to repeat a block of code.

  • while loop: Repeats as long as a condition is true. 
    i = 1
while i < 6:
    print(i)
    i += 1
  • for loop: Iterates over a sequence (like a list or a range of numbers). 
    fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Functions: Organizing Your Code

Functions are named, reusable blocks of code designed to perform a single, specific task. They make your code more organized, readable, and efficient. A function is defined with the def keyword.

Parameters and Arguments

def my_function(fname, age):  # fname and age are parameters
    print(fname + " is " + age + " years old.")

my_function("Teshome", "36")  # "Teshome" and "36" are arguments

Default Parameter Values

def my_function(country = "Ethiopia"):
    print("I am from " + country)

my_function("Sweden")  # Prints "I am from Sweden"
my_function()         # Prints "I am from Ethiopia"

Return Values

def calculate_square(x):
    return x * x

squared_number = calculate_square(5)
print(squared_number)  # Output: 25

Python Exercises

Beginner Exercise: Personalized Story Generator

  1. Ask the user for several pieces of information: their name, their city, the name of a friend, and an activity they enjoy (e.g., "playing football").
  2. Store each piece of information in a separate variable.
  3. Craft a short, one-paragraph story using the user's input.
  4. Use an f-string (formatted string) to easily embed the variables into your story.
  5. Example: 
    story = f"Once upon a time, {name} from {city} decided to go {activity} with their friend, {friend_name}."
print(story)
  6. Print the final, personalized story to the console.

Challenge Exercise: Student Gradebook

  1. Create an empty dictionary called gradebook. The keys of this dictionary will be student names (strings), and the values will be a list of their grades (integers). 
    gradebook = {"Abel": [88, 92], "Beti": [95]}
  2. Create three functions:
    • add_student(student_name): Checks if the student is already in the gradebook. If not, adds them with an empty list of grades.
    • add_grade(student_name, grade): Adds a new grade to the list of grades for a specific student. If the student doesn't exist, print an error message.
    • calculate_average(student_name): Finds the student, calculates the average of their grades, and prints it in a user-friendly format (e.g., "Abel's average grade is: 90.0"). Handle the case where a student has no grades yet.
  3. Write the main part of your program to test your functions:
    • Add 2-3 students to the gradebook.
    • Add several grades for each student.
    • Calculate and print the average for each student.
    • Try to add a grade for a student who doesn't exist to see your error message work.
