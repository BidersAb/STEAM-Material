Lesson 2: Introduction to Web Development

Welcome to the world of web development! This lesson will guide you through the three core technologies that power every single website you visit. We'll explore each one in detail and give you exercises to build your skills at every step.

HTML - The Architect of the Web

Think of HTML (HyperText Markup Language) as the architect's blueprint for a house. It defines the fundamental structure: where the rooms are, the doors, the windows, and the floors. In the same way, HTML lays out the structure of a webpage, telling the browser what is a heading, what is a paragraph, where an image should go, and what text should be a link.

The Anatomy of a Perfect HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Awesome Web Page</title>
</head>
<body>
</body>
</html>

HTML Tags

1. Text and Content Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <strong>: Bold/strong text
  • <em> or <i>: Italic/emphasized text
  • <a>: Anchor (hyperlink)
    Example: <a href="https://www.boeing.com" target="_blank">Visit Boeing</a>

2. Structural and Semantic Tags

  • <div>: Division container
  • <header>, <footer>, <main>, <nav>: Semantic containers

3. Image and List Tags

  • <img>: Image
    Example: <img src="ethiopian_logo.png" alt="Ethiopian Airlines Logo">
  • <ul> and <ol>: Unordered and Ordered lists with <li> items

HTML Exercise: My First Biography Page

  • Create biography.html
  • Include <title>, <header>, <main>, and <footer>
  • <header>: Use <h1> for name
  • <main>:
    • <h2>Early Life</h2> with <p>
    • <h2>Major Achievements</h2> with an <ol> of at least three items
    • <img> of the person
  • <footer>: Your name

CSS - The Interior Designer of the Web

CSS (Cascading Style Sheets) controls everything visual: colors, fonts, spacing, positioning, and animations.

How to Add CSS

  1. Internal CSS: Inside <style> tags in the <head>
  2. External CSS: Link to a .css file with <link>

CSS Selectors

  • Element selector: p { }
  • Class selector: .highlight { }
  • ID selector: #main-header { }

CSS Box Model

  • Content: Text or image
  • Padding: Space inside the element
  • Border: Line around the padding
  • Margin: Space outside the border

CSS Exercise: Style Your Biography Page

  • Use internal CSS (<style> in <head>)
  • Style the body background
  • Center <h1> and change its font
  • Style <footer> background
  • Give <img> a border
  • Set line-height for <p> tags to 1.5

JavaScript - The Magician of the Web

JavaScript is the programming language that brings interactivity to the web. It runs in the browser and responds to user actions.

How to Add JavaScript

Use <script> tags, usually placed before </body>.

<script>
  console.log("The page has loaded and the script is running!");
</script>

JavaScript Core Concepts

1. Variables

  • let: Reassignable variable
  • const: Constant (unchangeable)
  • var: Old version, use less

2. Data Types

  • String: "Hello"
  • Number: 42
  • Boolean: true or false
  • Array: ["a", "b", "c"]
  • Object: { name: "Alice", age: 25 }

3. Functions

<script>
  function calculateArea(width, height) {
    let area = width * height;
    return area;
  }

  let area = calculateArea(5, 10);
  console.log("The area is: " + area);
</script>

4. Conditional Logic

<script>
  let hour = 14;
  if (hour < 12) {
    console.log("Good morning!");
  } else if (hour < 18) {
    console.log("Good afternoon!");
  } else {
    console.log("Good evening!");
  }
</script>

DOM Manipulation

  • document.getElementById('id')
  • document.querySelector('.class')
  • .innerHTML, .textContent, .value
  • .style.property (e.g., backgroundColor)

JavaScript Exercise: Tip Calculator

  • HTML:
    • <h1>Tip Calculator</h1>
    • <input type="number" id="billAmount">
    • <button onclick="calculateTip()">Calculate 15% Tip</button>
    • <h2 id="tipResult"></h2>
  • JavaScript: 
    <script>
  function calculateTip() {
    let bill = parseFloat(document.getElementById('billAmount').value);
    let tip = bill * 0.15;
    let resultElement = document.getElementById('tipResult');
    resultElement.textContent = "Tip to leave: $" + tip.toFixed(2);
  }
</script>

Challenge: Mini To-Do List App

  • HTML:
    • <h1>My To-Do List</h1>
    • <input id="newItemInput">
    • <button onclick="addTask()">Add Task</button>
    • <ul id="taskList"></ul>
  • JavaScript: 
    <script>
  function addTask() {
    let taskText = document.getElementById('newItemInput').value;
    if (taskText === "") return;

    let li = document.createElement('li');
    li.textContent = taskText;

    li.onclick = function() {
      this.classList.toggle('completed');
    };

    document.getElementById('taskList').appendChild(li);
    document.getElementById('newItemInput').value = "";
  }
</script>
